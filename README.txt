## 📝 Mô tả dự án (Project Description)
Hệ thống Quản lý Nhân viên là giải pháp phần mềm tập trung nhằm tối ưu hóa việc quản trị dữ liệu nhân sự. Hệ thống hỗ trợ quản lý hồ sơ cá nhân, cấp phát tài khoản, phân quyền truy cập (RBAC) và xác thực người dùng bảo mật, giúp vận hành quy trình nhân sự chuyên nghiệp và an toàn.

## 🚀 Công nghệ sử dụng (Tech Stack)
* **Framework:** .NET Core 9.0 (Hiệu năng cao, bảo mật)
* **Database:** Microsoft SQL Server (Quản lý dữ liệu quan hệ)
* **Auth:** Cơ chế Xác thực & Phân quyền hệ thống
* **Tools:** SSMS, JSON Configuration, .NET CLI
